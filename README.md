# 个人刷题

注意：项目**编译无法通过**。

个人刷题习惯使用vscode，并且将java语言服务器的启动模式设置为 **LightWeight** 。

这样能减少很多报错提示，但是依旧有代码提示和补全。刷题十分轻便。

另外推荐使用注释样式插件(Better Comments)。可以辅助做笔记
